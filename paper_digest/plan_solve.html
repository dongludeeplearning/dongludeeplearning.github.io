Methodï¼ˆæ–¹æ³•ï¼‰

The Plan-and-Solve prompting method enhances zero-shot chain-of-thought reasoning through explicit planning and execution:

  1. Problem Understanding: Extract relevant variables and their corresponding numerals from the problem statement to ensure all necessary information is identified.
  2. Plan Formulation: Devise a complete solution plan by breaking down the problem into sequential subtasks that need to be addressed step-by-step.
  3. Plan Execution: Carry out the plan by calculating intermediate variables and performing operations according to the devised plan, with attention to numerical accuracy and commonsense reasoning.
  4. Enhanced Instructions (PS+): Add detailed guidance for variable extraction, intermediate result calculation, and attention to both numerical computation and commonsense reasoning.
  5. Answer Extraction: Use a separate prompting step to extract the final numerical or option-based answer from the generated reasoning text.

Why it Works?

Plan-and-Solve prompting succeeds by addressing the fundamental limitations of Zero-shot-CoT through structured reasoning:

  • Explicit Planning: By requiring LLMs to first devise a complete solution plan, the method ensures all necessary reasoning steps are identified upfront, reducing missing-step errors.
  • Step-by-Step Execution: The execution phase enforces systematic progression through the plan, ensuring each subtask is addressed in the correct sequence with proper intermediate calculations.
  • Variable Awareness: Explicit extraction and tracking of relevant variables and numerals ensures LLMs maintain focus on key problem elements throughout the reasoning process.
  • Error Mitigation: Detailed instructions for calculation and commonsense reasoning help LLMs avoid both computational mistakes and semantic misunderstandings.
  • Structured Reasoning: The plan-and-execute framework provides a clear structure that guides LLMs toward more coherent and complete reasoning processes.

Conclusionï¼ˆç»“è®ºï¼‰

This influential work introduces Plan-and-Solve prompting, a significant advancement in zero-shot chain-of-thought reasoning that addresses the critical limitations of standard Zero-shot-CoT. By guiding large language models to first devise comprehensive solution plans and then execute them step-by-step with explicit attention to variable extraction, intermediate calculations, and commonsense reasoning, PS+ prompting achieves substantial performance improvements across diverse reasoning benchmarks. The method's elegance lies in its simplicity and zero-shot nature, requiring no manual demonstrations while consistently outperforming established baselines including Zero-shot-CoT, Zero-shot-PoT, and even Manual-CoT on several datasets. Experimental results demonstrate remarkable gains on arithmetic reasoning tasks (76.7% average accuracy) and strong performance on commonsense and symbolic reasoning, with PS+ prompting reducing critical error types by 2-3% absolute. Through systematic error analysis and correlation studies, the authors provide compelling evidence that structured planning and careful execution are key to unlocking more reliable reasoning capabilities in large language models. This work not only advances the state-of-the-art in prompt engineering but also provides valuable insights into how explicit planning instructions can mitigate the inherent limitations of current language model architectures. The approach represents a promising direction for developing more robust and interpretable reasoning systems, with potential applications extending beyond mathematical and logical reasoning to complex decision-making and problem-solving domains.

# Plan-and-Solve Prompting Implementation

// Zero-shot PS+ prompting with plan formulation and execution
def plan_solve_prompt(question):
    """Implement Plan-and-Solve prompting strategy"""
    
    # PS+ Prompt with detailed instructions
    ps_plus_prompt = f"""Q: {question}
A: Let's first understand the problem, extract relevant variables and their corresponding numerals, and make a complete plan. Then, let's carry out the plan, calculate intermediate variables (pay attention to correct numerical calculation and commonsense), solve the problem step by step, and show the answer."""
    
    # Generate reasoning with plan and execution
    reasoning_text = query_language_model(ps_plus_prompt)
    
    # Extract final answer using separate prompt
    extraction_prompt = f"""{reasoning_text}
Therefore, the answer (arabic numerals) is"""
    final_answer = query_language_model(extraction_prompt)
    
    return final_answer
