Methodï¼ˆæ–¹æ³•ï¼‰

Zolly addresses perspective distortion through a new camera model and distortion-aware representation:

  1. Distortion Image Representation: Introduces a novel 2D representation termed distortion image, which describes the 2D dense distortion scale of the human body. The distortion scale is analyzed based on the relative location of the human body to the camera center, which is the main cause of perspective distortion.
  2. Distance Estimation from Distortion: Estimates the distance from the camera to the human body using distortion scale features extracted from the distortion image, rather than relying on background environment context features which may not be reliable.
  3. Distortion-Aware Mesh Reconstruction: Integrates the distortion feature with image features to reconstruct the body mesh, enabling the model to account for perspective effects during mesh recovery.
  4. Dual Projection Loss: Simultaneously uses perspective and weak-perspective projection loss to formulate the correct projection matrix and locate the human body position accurately. This ensures both accurate 3D pose estimation and proper 2D alignment.
  5. PDHuman Dataset: Contributes the first synthetic dataset PDHuman specifically for perspective-distorted human images, and extends two real-world datasets to provide diverse training data with varying degrees of perspective distortion.

Discussionï¼ˆè®¨è®ºï¼‰

Why it Works?

Zolly succeeds by explicitly modeling perspective distortion and using distortion-aware features:

  • Distortion Image Representation: By introducing the distortion image as a 2D dense representation of distortion scale, the method explicitly captures perspective effects caused by the relative location of the human body to the camera center, which is the fundamental cause of perspective distortion.
  • Distortion-Based Distance Estimation: Estimating distance from distortion scale features is more reliable than using background environment context, as distortion scales directly reflect the geometric relationship between the camera and human body.
  • Feature Integration: Integrating distortion features with image features enables the pose estimator to account for perspective effects during mesh reconstruction, improving accuracy for close-range images with strong distortion.
  • Dual Projection Supervision: Using both perspective and weak-perspective projection loss ensures the model learns correct projection matrices while maintaining flexibility for different scenarios, achieving both accurate 3D pose and proper 2D alignment.
  • Diverse Training Data: The PDHuman synthetic dataset and extended real-world datasets provide diverse perspective-distorted scenarios, enabling robust learning of distortion-aware reconstruction.

Conclusionï¼ˆç»“è®ºï¼‰

This pioneering work introduces Zolly, the first 3D human mesh reconstruction method specifically designed to handle perspective-distorted images. By recognizing that perspective distortion is mainly caused by the relative location of the human body to the camera center, the method introduces a novel distortion image representation to explicitly model 2D dense distortion scales. Through distortion-based distance estimation and distortion-aware mesh reconstruction, Zolly accurately estimates focal length and formulates correct projection matrices for perspective-distorted scenarios. The dual projection loss framework, combining perspective and weak-perspective supervision, ensures both accurate 3D pose estimation and proper 2D alignment. Extensive experiments demonstrate Zolly's superior performance over existing state-of-the-art methods on both perspective-distorted datasets and the standard benchmark (3DPW). The contribution of PDHuman, the first synthetic dataset for perspective-distorted human images, along with extended real-world datasets, addresses the critical lack of training data for this challenging scenario. This work opens new possibilities for accurate 3D human mesh reconstruction in close-range scenarios where perspective distortion cannot be ignored, such as video conferencing, virtual try-on, and interactive applications. Zolly's approach to explicitly modeling and leveraging perspective distortion represents a significant advancement in handling real-world 3DHMR challenges where naive focal length assumptions fail.

// Distortion image generation
def generate_distortion_image(image, body_bbox):
    """Generate 2D dense distortion scale representation"""
    # Analyze relative location of human body to camera center
    distortion_features = extract_distortion_scale(image, body_bbox)
    distortion_img = distortion_encoder(distortion_features)
    return distortion_img

// Distance estimation from distortion
def estimate_distance_from_distortion(distortion_img):
    """Estimate camera-to-person distance using distortion scale features"""
    distortion_features = extract_distortion_features(distortion_img)
    distance = distance_estimator(distortion_features)
    return distance

// Distortion-aware mesh reconstruction
def reconstruct_mesh_with_distortion(image, distortion_img, distance):
    """Reconstruct 3D mesh integrating distortion and image features"""
    image_features = extract_image_features(image)
    distortion_features = extract_distortion_features(distortion_img)
    # Integrate distortion with image features
    combined_features = integrate_features(image_features, distortion_features, distance)
    mesh_params = mesh_estimator(combined_features)
    return mesh_params

// Dual projection loss
def dual_projection_loss(mesh_3d, keypoints_2d, focal_length, camera_params):
    """Compute both perspective and weak-perspective projection loss"""
    # Perspective projection loss
    proj_persp = perspective_projection(mesh_3d, focal_length, camera_params)
    loss_persp = compute_loss(proj_persp, keypoints_2d)
    
    # Weak-perspective projection loss
    proj_weak = weak_perspective_projection(mesh_3d, camera_params)
    loss_weak = compute_loss(proj_weak, keypoints_2d)
    
    return loss_persp + loss_weak

// Main pipeline
def zolly_pipeline(image, body_bbox):
    """Complete Zolly pipeline for perspective-distorted mesh recovery"""
    # Step 1: Generate distortion image
    distortion_img = generate_distortion_image(image, body_bbox)
    
    # Step 2: Estimate distance from distortion
    distance = estimate_distance_from_distortion(distortion_img)
    
    # Step 3: Reconstruct mesh with distortion awareness
    mesh_3d = reconstruct_mesh_with_distortion(image, distortion_img, distance)
    
    return mesh_3d, distance
