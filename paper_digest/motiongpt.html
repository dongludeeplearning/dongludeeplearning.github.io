Paper Digest • NeurIPS 2023

MotionGPT: Human Motion as a Foreign Language

Biao Jiang, Xin Chen, Wen Liu, Jingyi Yu, Gang Yu, Tao Chen
Fudan University & Tencent PCG
One-Sentence Problem: How to build a unified model that handles multiple motion-related tasks (generation, captioning, prediction, etc.) by treating human motion as a foreign language?
一句话问题：如何通过将人体运动视为一门外语，构建一个处理多种运动相关任务（生成、描述、预测等）的统一模型？

Contributions（贡献）

  • Unified Motion-Language Model: Proposed MotionGPT, a unified pre-trained model capable of handling diverse motion tasks (text-to-motion, motion-to-text, prediction, in-betweening) within a single framework.
    统一的运动-语言模型：提出了 MotionGPT，这是一个统一的预训练模型，能够在单个框架内处理多种运动任务（文本到运动、运动到文本、预测、中间帧生成）。
  • Motion Vocabulary: Introduced a motion tokenizer based on VQ-VAE to discretize 3D motion into motion tokens, effectively treating motion as a "foreign language".
    运动词汇表：引入了基于 VQ-VAE 的运动分词器，将 3D 运动离散化为运动 token，有效地将运动视为一种“外语”。
  • Prompt-Based Instruction Tuning: Utilized prompt learning to unify different tasks into a question-answering format, enabling the model to learn from mixed motion-language data.
    基于 Prompt 的指令微调：利用 prompt 学习将不同任务统一为问答格式，使模型能够从混合的运动-语言数据中学习。

Intro Figure（示例）

Challenges（挑战）

  • Multi-Modal Alignment: Aligning the semantic space of human motion (continuous 3D data) with natural language (discrete tokens) is difficult.
    多模态对齐：将人体运动（连续 3D 数据）的语义空间与自然语言（离散 token）对齐是困难的。
  • Task Diversity: Different motion tasks (e.g., generation vs. captioning) have different inputs and outputs, making it hard to solve them with a single model architecture.
    任务多样性：不同的运动任务（例如生成 vs. 描述）具有不同的输入和输出，难以用单一的模型架构来解决。

Method（解决方法）

  1. Motion Tokenizer (VQ-VAE): Train a VQ-VAE on large-scale motion data to convert continuous motion sequences into discrete codebook indices (tokens).
    运动分词器 (VQ-VAE)：在大规模运动数据上训练 VQ-VAE，将连续运动序列转换为离散的码本索引（token）。
  2. Motion-Language Pre-training: Use a T5-based architecture to model the probability of next tokens (both text and motion) in a unified vocabulary space.
    运动-语言预训练：使用基于 T5 的架构在统一的词汇空间中对下一个 token（文本和运动）的概率进行建模。
  3. Instruction Tuning: Formulate various tasks as prompt-based instructions (e.g., "Generate motion: [Text]") and fine-tune the model to follow these instructions.
    指令微调：将各种任务制定为基于 prompt 的指令（例如，“生成动作：[文本]”），并微调模型以遵循这些指令。

Overview Figure（示意图）

Experiments（实验）

  • Datasets: Evaluated on HumanML3D and KIT-ML datasets, covering text-motion paired data.
    数据集：在 HumanML3D 和 KIT-ML 数据集上进行评估，涵盖了文本-运动配对数据。
  • Text-to-Motion: Achieved SOTA results in FID, R-Precision, and Diversity metrics, outperforming diffusion-based and other transformer-based baselines.
    文本生成运动：在 FID、R-Precision 和多样性指标上取得了 SOTA 结果，优于基于扩散和其他基于 Transformer 的基线。
  • Motion Captioning: Demonstrated superior performance in generating descriptive text for given motion sequences (high BLEU and ROUGE scores).
    运动描述：在为给定的运动序列生成描述性文本方面表现出优越的性能（高 BLEU 和 ROUGE 分数）。

Discussion（讨论）

  • Unification Benefit: The "Motion as Language" paradigm allows the model to leverage the vast knowledge and reasoning capabilities inherent in large language models (LLMs) for motion understanding.
    统一的好处： “运动即语言”的范式允许模型利用大型语言模型 (LLMs) 固有的丰富知识和推理能力来进行运动理解。
  • Zero-Shot Capability: The instruction tuning approach enables the model to generalize to unseen tasks or prompts, demonstrating strong zero-shot capabilities.
    零样本能力： 指令微调方法使模型能够泛化到未见过的任务或提示，展示了强大的零样本能力。

High-Level Insights: Why it Works?

MotionGPT works by bridging the gap between two different modalities (motion and text) through a shared discrete representation space. By discretizing motion into tokens, it transforms continuous motion generation into a sequence-to-sequence problem, which Transformers excel at. The unified vocabulary allows the model to learn joint distributions of text and motion, enabling bidirectional generation and understanding within a single model.
MotionGPT 之所以有效，是因为它通过共享的离散表示空间弥合了两种不同模态（运动和文本）之间的差距。通过将运动离散化为 token，它将连续运动生成转换为序列到序列的问题，而这正是 Transformer 擅长的。统一的词汇表允许模型学习文本和运动的联合分布，从而在单个模型中实现双向生成和理解。

Conclusion（结论）

  • MotionGPT successfully unifies human motion and language into a single pre-trained model.
    MotionGPT 成功地将人体运动和语言统一到一个单一的预训练模型中。
  • It achieves state-of-the-art performance on diverse motion tasks, showing the effectiveness of the discrete motion token approach.
    它在多种运动任务上取得了最先进的性能，展示了离散运动 token 方法的有效性。
  • The work highlights the potential of treating motion as a foreign language for multi-modal understanding and generation.
    这项工作突出了将运动视为一种外语用于多模态理解和生成的潜力。

class MotionGPT(nn.Module):
    def forward(self, input_ids, labels):
        # input_ids contains both text tokens and motion tokens
        # T5Stack processes them in a shared embedding space
        outputs = self.t5_model(
            input_ids=input_ids,
            labels=labels
        )
        return outputs.loss

    def generate(self, text_prompt):
        # 1. Tokenize text prompt
        text_tokens = self.tokenizer(text_prompt)
        
        # 2. Autoregressive generation of motion tokens
        motion_tokens = self.t5_model.generate(text_tokens)
        
        # 3. Decode motion tokens back to 3D motion using VQ-VAE decoder
        motion = self.vq_decoder(motion_tokens)
        return motion
