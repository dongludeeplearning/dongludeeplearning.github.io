High-Level Insights: Why it Works?

VOCA works by simplifying the problem: instead of predicting absolute vertex positions, it predicts displacements from a template. By conditioning on a "speaker ID", it learns a manifold of speaking styles. The use of DeepSpeech features ensures that the input to the network contains rich phonetic information, making the mapping to mouth movements easier to learn than from raw audio.

# VOCA Inference Flow
def forward(audio, template, speaker_id):
    # 1. Extract Audio Features
    audio_feat = deepspeech_model(audio)
    
    # 2. Encode Audio + Speaker Style
    # One-hot speaker ID concatenated with audio window
    encoding = encoder(audio_feat, speaker_id)
    
    # 3. Decode to Vertex Offsets
    offsets = decoder(encoding)
    
    # 4. Apply to Template
    animated_mesh = template + offsets
    return animated_mesh
