One-Sentence Problem: How to generate high-fidelity, long-horizon full-body human interactions (including navigation and dexterous manipulation) within complex dynamic object and static scene environments?
一句话问题：如何在复杂的动态物体和静态场景环境中，生成高保真、长时程的全身交互（包括导航和灵巧操作）？

Contributions（贡献）

  • Hierarchical Framework: Proposed HOSIG, a novel framework that decouples interaction into scene-aware grasping, heuristic navigation, and motion diffusion.
    分层框架：提出了 HOSIG，一种新颖的框架，将交互解耦为场景感知抓取、启发式导航和运动扩散。
  • Unified Navigation & Manipulation: Bridged the gap between scene-aware navigation and dexterous object manipulation, enabling unlimited motion length via autoregressive generation.
    统一导航与操作：弥合了场景感知导航和灵巧物体操作之间的差距，通过自回归生成支持无限的运动长度。
  • High Fidelity: Achieved collision-free postures with precise hand-object contact and finger-level accuracy in trajectory-controlled motions.
    高保真度：在轨迹控制运动中实现了无碰撞姿态，并具有精确的手-物接触和指尖级精度。

Challenges（挑战）

  • Scene Context Neglect: Existing HOI methods often ignore scene context, causing penetrations, while HSI methods struggle with fine-grained manipulation.
    忽视场景上下文：现有的 HOI 方法通常忽略场景上下文，导致穿透，而 HSI 方法难以处理精细操作。
  • Coordination Complexity: Coordinating long-range navigation (obstacle avoidance) with fine-grained object manipulation (dexterous grasping) is difficult.
    协调复杂性：协调长距离导航（避障）与精细物体操作（灵巧抓取）非常困难。

Method（解决方法）

  1. Scene-Aware Grasp Generator: Generates collision-free whole-body postures with precise hand-object contact by integrating local geometry constraints.
    场景感知抓取生成器：通过整合局部几何约束，生成具有精确手-物接触的无碰撞全身姿态。
  2. Heuristic Navigation: Plans obstacle-avoiding paths in complex indoor environments using compressed 2D floor maps and dual-component spatial reasoning.
    启发式导航：使用压缩的 2D 平面图和双组件空间推理，在复杂的室内环境中规划避障路径。
  3. Scene-Guided Motion Diffusion: Generates trajectory-controlled, full-body motions with finger-level accuracy using spatial anchors and dual-space classifier-free guidance.
    场景引导运动扩散：使用空间锚点和双空间无分类器引导，生成具有指尖级精度的轨迹控制全身运动。

Method Explanation (Detailed)

Overview: HOSIG employs a hierarchical approach to synthesize full-body interactions, ensuring both local dexterity and global navigation capability.
概述： HOSIG 采用分层方法合成全身交互，确保局部灵活性和全局导航能力。
  • 1. Scene-Aware Grasp Pose Generation:
    This module focuses on generating a static, collision-free full-body pose for grasping. It integrates local object geometry to ensure precise hand-object contact while accounting for the surrounding scene to avoid collisions. This provides a high-quality target state for the motion generation.
    1. 场景感知抓取姿态生成：
    该模块专注于生成用于抓取的静态、无碰撞全身姿态。它整合了局部物体几何结构以确保精确的手-物接触，同时考虑周围场景以避免碰撞。这为运动生成提供了高质量的目标状态。
  • 2. Heuristic Navigation on 2D Obstacle-Aware Map:
    To handle complex indoor environments, the system projects the 3D scene into a 2D occupancy map. A heuristic path planning algorithm (like A*) is then used to find a collision-free trajectory from the start point to the interaction target, ensuring long-range navigation capability.
    2. 二维障碍感知地图上的启发式导航：
    为了处理复杂的室内环境，系统将 3D 场景投影为 2D 占用地图。然后使用启发式路径规划算法（如 A*）找到从起点到交互目标的无碰撞轨迹，确保长距离导航能力。
  • 3. Scene-Guided Controllable Motion Generation:
    This is the core motion synthesis engine. It uses a diffusion model conditioned on the planned trajectory and the target grasp pose. It generates smooth, natural transitions from locomotion to manipulation, handling the coordination of full-body movements.
    3. 场景引导的可控运动生成：
    这是核心运动合成引擎。它使用以规划轨迹和目标抓取姿态为条件的扩散模型。它生成从移动到操作的平滑、自然过渡，处理全身运动的协调。
  • 4. Joint & Scene Guidance:
    To improve generation quality, the model uses dual-space classifier-free guidance. "Joint Guidance" ensures the motion adheres to biomechanical constraints and the target pose, while "Scene Guidance" ensures the motion remains collision-free with the static environment throughout the sequence.
    4. 关节与场景引导：
    为了提高生成质量，模型使用双空间无分类器引导。“关节引导”确保运动符合生物力学约束和目标姿态，而“场景引导”确保运动在整个序列中与静态环境保持无碰撞。

Discussion（讨论）

  • Hierarchical Perception: The key to success lies in the hierarchical perception of the scene, allowing for focused processing of local geometry for grasping and global layout for navigation.
    分层感知： 成功的关键在于对场景的分层感知，允许针对抓取处理局部几何结构，针对导航处理全局布局。
  • Dual-Space Guidance: The use of dual-space classifier-free guidance in the diffusion model effectively balances the constraints of trajectory control and interaction fidelity.
    双空间引导： 在扩散模型中使用双空间无分类器引导有效地平衡了轨迹控制和交互保真度的约束。

High-Level Insights: Why it Works?

HOSIG works by breaking down the complex problem of full-body interaction into manageable sub-problems: static pose generation, path planning, and dynamic motion synthesis. By solving these sequentially and integrating them with scene-aware constraints, it avoids the pitfalls of end-to-end approaches that often fail to capture both global navigation and local manipulation details simultaneously.
HOSIG 通过将全身交互的复杂问题分解为可管理的子问题：静态姿态生成、路径规划和动态运动合成。通过顺序解决这些问题并将其与场景感知约束集成，它避免了端到端方法通常无法同时捕捉全局导航和局部操作细节的缺陷。

import heapq
import numpy as np

def create_occupancy_map(point_cloud, grid_size, map_size):
    # Project 3D point cloud to 2D occupancy grid
    width, height = map_size
    occupancy_map = np.zeros((height, width), dtype=np.int8)
    for point in point_cloud:
        x, y, z = point
        grid_x, grid_y = int(x // grid_size), int(y // grid_size)
        if 0 <= grid_x < width and 0 <= grid_y < height:
            occupancy_map[grid_y, grid_x] = 1  # Mark as occupied
    return occupancy_map
